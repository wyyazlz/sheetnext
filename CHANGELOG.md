# Changelog

## [0.2.0] - 2026-02-28

### Highlights

First full open-source release with complete source code and comprehensive spreadsheet capabilities.

### Added

- **Full Source Code** — Open edition ships with complete, readable source code
- **Formula Engine** — math, statistics, lookup, text, date, logical
- **Conditional Formatting** — Cell rules, color scales, data bars, icon sets, top/bottom rules, custom expressions
- **AutoFilter & Sorting** — Column filtering with multi-condition support and custom sort dialogs
- **Super Tables** — Structured references, auto-expand, built-in styles, calculated columns
- **Slicers** — Interactive slicer controls for tables
- **Sparklines** — In-cell mini charts (line, column, win/loss)
- **Data Validation** — Drop-down lists, number/date/text constraints, custom formula rules
- **Sheet Protection** — Cell-level locking, password protection, granular permissions
- **Comments** — Cell comments with threaded replies
- **Find & Replace** — Full-featured dialog with regex and scope options
- **Freeze Panes** — Row and column freezing with split-view
- **Undo/Redo** — Full operation history
- **Rich Text Editing** — In-cell rich text with font, color, and style formatting
- **Canvas Rendering** — High-performance canvas-based virtual scrolling
- **XLSX / CSV / JSON** — Native file import & export (getData API for open edition)
- **I18n** — English (en-US) and Chinese (zh-CN) locale support
- **API Documentation** — Full reference in `docs-detail.md` for AI-driven development

### Changed

- Restructured project for open edition
- Improved toolbar and UI layout
